Google Tag Manager Template

Jelikož jsme v praxi zjistili, že většina template poskytovaných přímo výrobci jednotlivých CMP nástrojů je tak trochu nedostatečná, rozhodli jsme si ji vyrobit sami.

Do vlastního GTM ji přidáte jednoduše pomocí importu kontejneru. Jakmile tento proces dokončíte, budete mít k dispozici jeden nový tag a jednu novou proměnnou.
Pomocí tagu si mužete nastavit libovolnou skupinu souhlasů (včetně vlastních) na základě libolvoných dat/proměnných, které máte k dispozici.

Template je náhradou jinak potřebné implementace volání gtag příkazů consent-default a consent-update


window.dataLayer = window.dataLayer || [];
function gtag(){dataLayer.push(arguments);}

gtag('consent', 'default', {
  'ad_storage': 'denied',
  'analytics_storage': 'denied'
});

S jeho pomocí můžete nastavit všechny dostupné funkce:

  • Default
    • ad_storage
    • analytics_storage
    • functionality_storage
    • personalization_storage
    • security_storage
    • libovolný vlastní typ souhlasu

    • Regionální souhlasy
    • Wait For Update nastavení
  • Update
    • ad_storage
    • analytics_storage
    • functionality_storage
    • personalization_storage
    • security_storage
    • libovolný vlastní typ souhlasu
